# Code Of Conduct

Image Cockpit for Codex Workflows is intended to be a practical, respectful project space for local image production tooling.

## Expected Behavior

- Be clear, specific, and constructive.
- Assume good faith while still naming risks plainly.
- Respect creators' rights and avoid submitting license-unclear assets.
- Keep discussions focused on the product boundary, workflows, and user outcomes.
- Help make the four primary workflows easier to understand.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Sharing secrets, private data, or credential material.
- Submitting model weights, copyrighted assets without permission, or unclear third-party content.
- Pressuring maintainers to add direct API calls, public releases, or main merges without explicit approval.

## Enforcement

Maintainers may edit, hide, or remove content that violates this code, and may limit participation when needed to keep the project workable and respectful.
