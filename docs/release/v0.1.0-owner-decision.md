## Verified For Private MVP Review

- Guided Start exposes the four primary workflows: image generation, image editing, sprite sheet generation, and sprite sheet editing.
- The app writes local Codex handoff jobs and does not require an OpenAI API key.
- Image generation handoff jobs stay prompt-focused.
- Image editing handoff jobs include selected source assets, edit notes, and annotations.
- Sprite generation and sprite editing handoff jobs include grid, action, and frame count context.
- The Codex job button locks during pending work and unlocks on failure, unavailable, completed-without-image, or orphaned unknown status.
- Manual handoff is documented and Browser-QA verified with an outbox image imported through `Import Latest`.
- Japanese / English switching covers the core workflow, canvas, sprite, and export labels needed for the MVP.
- `npm run verify` passes locally, and GitHub Actions runs the same release-check path.
