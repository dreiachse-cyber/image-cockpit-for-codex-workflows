## Verification Commands

The public baseline verification path is:

```powershell
npm run verify
```

Expanded verification path:

```powershell
npm run doctor
npm run typecheck
npm test
npm run build
npm run smoke
npm run release:audit
```

Optional browser review smoke:

```powershell
npm run ui:smoke
```

Optional real Codex runner smoke when a runnable local Codex CLI is available:

```powershell
npm run codex:smoke
```

Full owner-review local sweep on a Codex-installed review machine:

```powershell
npm run review:local
```

Expected result: all commands pass. `npm run doctor` may warn that the Codex executable is unavailable in environments where Windows blocks `codex exec`; that is acceptable when manual handoff is being reviewed. On this Windows review environment, `npm run doctor` selects the terminal-runnable Codex CLI under `%LOCALAPPDATA%\OpenAI\Codex\bin\...` and verifies `--help` successfully.

## Do Not Treat As Accepted

- Direct OpenAI API integration from the app
- Required `OPENAI_API_KEY` setup
- ComfyUI, AUTOMATIC1111, Replicate, or other external adapters
- IndexedDB persistence
- Game-engine-specific exports
- Bundled model weights or license-unclear assets
