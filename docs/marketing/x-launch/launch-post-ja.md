# X Launch Post JA

Codex時代の画像制作パイプラインUIとして、Image Cockpit for Codex Workflows を公開しました。

AI画像生成UIそのものというより、Codexで作った画像を人間が確認、注釈、修正して、ピクセルアートやアニメーション素材へ育てるためのローカル制作コクピットです。

- ピクセルアート生成
- 画像編集メモと矩形注釈
- 生成/アップロード画像からのアニメーション生成
- GIF / WebP / sprite sheet / animation pack export

アプリ自体はOpenAI公式ではなく、OpenAI APIキーも不要です。ローカルのCodex handoff / `codex exec` workflowを使います。

https://github.com/dreiachse-cyber/image-cockpit-for-codex-workflows
