## Posting Checks

- Do not include API keys, tokens, `codex-handoff/` job logs, local absolute paths, or private prompt dumps.
- Prefer screenshots that show the workflow, not browser chrome or personal bookmarks.
- Mention that the app is unofficial and not affiliated with OpenAI.
- Mention that the app itself does not require an OpenAI API key and does not call OpenAI APIs directly.
- Mention that the intended route is local Codex handoff / `codex exec`.
- Keep the message focused on pixel art generation, image editing notes, and animation generation.
