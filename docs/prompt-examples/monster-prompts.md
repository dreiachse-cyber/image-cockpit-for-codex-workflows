# Monster Pixel-Art Prompt Examples

作成日: 2026-06-26

このファイルは、モンスター系ピクセルアート生成prompt例です。
ゲーム素材として扱いやすいように、1体のみ・全身・中央配置・背景なし・読みやすいシルエットを基本条件にしています。

## Common Negative Prompt

```text
blur, text, watermark, logo, cropped body, cut off body, extra duplicate monster, scenery, detailed background, floor shadow, photorealistic, 3d render, vector art, gore
```
